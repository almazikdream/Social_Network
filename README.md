I'm learning Java Spring Boot, and my first task was to create a pet project called Social Network.
I tried to use all four of the OOP principles:

1. Encapsulation - is the encapsulation of fields and functions that work with this data into a single class, and hiding internal implementation details from the outside world.

2. Inheritance - is a mechanism for creating new classes based on existing ones, allowing them to get their properties and methods.

3. Polymorphism - is the ability of objects from different classes to be treated equally through a common base class or interface.

4. Abstraction - is highlighting key characteristics of an object while ignoring minor details

It was an interesting expirience and I'm not going to stop, and will keep moving forward.
